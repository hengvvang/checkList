## 七、内置 Vim 插件功能（无需额外安装）
Zed Vim 模式默认集成了多个常用 Vim 插件的核心能力，开箱即用。

### 7.1 Surround 环绕操作
用于快速给文本添加/修改/删除包裹符号（括号、引号、标签等）。
| 快捷键 | 功能说明 |
|--------|----------|
| `ys + 文本对象 + 符号` | 添加环绕，例如 `ysiw"` 给单词加双引号 |
| `cs + 旧符号 + 新符号` | 修改环绕，例如 `cs"'` 把双引号改成单引号 |
| `ds + 符号` | 删除环绕，例如 `ds(` 删除括号 |
| 可视模式下 `S + 符号` | 给选中内容添加环绕（需手动开启配置） |

### 7.2 Comment 注释
| 快捷键 | 功能说明 |
|--------|----------|
| `gcc` | 切换当前行的行注释 |

### 7.3 其他内置插件能力

## 八、常用自定义配置参考
Zed Vim 模式支持高度自定义，以下是高频的配置项，可在 `keymap.json` 和 `settings.json` 中添加。

1. **jk 快速退出插入模式**
```json
{
  "context": "vim_mode == insert",
  "bindings": {
    "j k": "vim::NormalBefore"
  }
}
```

2. **开启子词跳转（驼峰/蛇形命名拆分）**
```json
{
  "context": "VimControl && !menu && vim_mode != operator",
  "bindings": {
    "w": "vim::NextSubwordStart",
    "b": "vim::PreviousSubwordStart",
    "e": "vim::NextSubwordEnd",
    "g e": "vim::PreviousSubwordEnd"
  }
}
```

3. **常用行为设置（settings.json）**
```json
{
  "vim": {
    "use_system_clipboard": "always",
    "use_smartcase_find": true,
    "toggle_relative_line_numbers": true,
    "highlight_on_yank_duration": 200
  }
}
```
